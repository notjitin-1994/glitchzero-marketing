# **App Name**: Digital Disha

## Core Features:

- Service Showcase: Visually present GlitchZero's app development services with engaging descriptions tailored for small business owners.
- Platform Compatibility Display: Clearly display all supported platforms (iOS, Android, MacOS, Windows, web, etc.) with corresponding icons.
- Customization Highlights: Emphasize that all applications are custom-built and tailored to the specific needs of Indian small businesses.
- Client Testimonials: Feature short, impactful testimonials from satisfied clients (specifically other Indian small businesses).
- Lead Capture Form: A simple form to collect potential client contact information for personalized follow-ups.
- Localized Content Tool: Use a generative AI tool to translate website content into common Indian languages.

## Style Guidelines:

- Background: Obsidian (#121212) for a professional, 'void' feel, creating depth.
- Surfaces: Carbon (#1C1C1E) used for cards, panels, and form backgrounds to provide contrast and highlight content.
- Text (Primary): Platinum (#E5E5E7) used for headings and primary content to ensure high readability.
- Text (Secondary): Tungsten (#8E8E93) used for metadata, descriptions, and secondary information.
- Accent: Signal Orange (#FF4F00) used for primary call-to-action buttons, hover states, and key interactive elements to draw attention.
- Success: Terminal Green (#30D158) subtly integrated for positive feedback and status indicators, primarily within the tech demo section.
- Main Headlines: 'Manrope' ExtraBold for strong, all-caps headings with +2% letter spacing.
- Subheadlines: 'Manrope' Medium for clear, sentence-case subheadings.
- Body Copy: 'Manrope' Regular for easy-to-read body text in sentence case.
- Tech Specs / UI Tags: 'JetBrains Mono' Medium in uppercase with +5% letter spacing.
- Code Snippets: 'JetBrains Mono' Regular in lowercase for realistic code representation.
- Use simple, outlined icons in Signal Orange (#FF4F00) to represent services and platforms. Icons should be easily understandable by a non-technical audience.
- Implement a clean, single-page layout using a grid system. Prioritize mobile responsiveness for Indian users.
- Subtle animations on scroll and hover, such as a gentle fade-in or a slight scaling effect on cards, to enhance the user experience without being distracting.
